Why remote attendance

為什麼遠端團隊需要專屬的打卡系統？

過去打卡這件事很單純：員工早上到辦公室刷卡、晚上下班再刷一次，HR 月底拉報表結薪水。 但遠端工作 (Remote Work) 與混合辦公 (Hybrid Work) 普及後，傳統打卡方式碰到三個明顯斷點：

  • 實體打卡機 / 公司 Wi-Fi 限制 — 員工在家工作根本打不到卡。
  • 員工自填工時 — 主管不知道誰真的有上班、HR 不知道工時是否正確。
  • Office 與 Remote 工時混在一起 — 月底結薪、人力統計時無法區分。

遠端打卡系統解決的不是「監控員工」，而是讓工時資料自動化、可信、可追溯。 員工不必每天在群組裡發「打卡了」，主管不必追進度，HR 不必對單，會計也不必猜這個月誰請了幾天假。

How it works

ClocDot 怎麼解決遠端打卡的三個難題

難題一：怎麼確認員工真的有上班？

GPS 自動記錄打卡座標，並比對辦公室地點，自動標記為 Office 或 Remote。 員工不必主動回報，主管不必追問，工時資料源於系統而非自填，可信度高。

難題二：跨平台、跨裝置怎麼裝？

ClocDot 是 PWA，瀏覽器開啟就能加到桌面，iOS / Android / Windows / Mac 通用。 不必上架 App Store、不必員工各自裝 App。連線斷了也能離線打卡，恢復後自動同步。

難題三：月底結薪資怎麼結？

每日明細含 Office / Remote 標記、扣完午休的實際工時、假別與加班。月底一鍵匯出 CSV， Excel 直接開、繁中不亂碼，會計不用對單，HR 不用算特休。

Who it's for

哪些團隊適合用遠端打卡系統？

🌍 全遠端 (Fully Remote) 團隊

公司沒有實體辦公室，或員工散佈各地。GPS 打卡可記錄員工實際所在城市，方便異地差勤管理。

🏢 混合辦公 (Hybrid) 團隊

員工部分時間進辦公室、部分時間在家。ClocDot 自動分辨並分流統計，不必員工自己標註。

✈️ 跨時區 / 跨國團隊

員工分散在不同國家或時區。系統以員工本地時間記錄、彙整時可切換時區呈現。

🛵 外勤 / 業務團隊

業務、外勤、工程師到客戶端服務時無法刷卡。手機 GPS 打卡可記錄拜訪地點與時間。

🚀 新創 / 小團隊

5 人以下永久免費，不需要 HR 系統那種複雜設定。審核後就能讓全團隊開始打卡。

FAQ

關於遠端打卡的常見問題

遠端打卡和傳統打卡有什麼不同？

傳統打卡綁定實體機器或公司網路，員工必須回到辦公室才能打卡。遠端打卡透過手機 GPS 或 PWA 進行，無論員工在家、咖啡廳、外地出差或跨國工作，都能完成打卡並自動標記地點來源（Office / Remote）。

員工在家打卡會不會作弊？

ClocDot 會記錄打卡當下的 GPS 座標與時間戳記，並自動比對是否在公司範圍內。所有打卡記錄都會保留地點軌跡，主管可在後台檢視。重點不是「監控員工」，而是讓 Remote / Office 工時清楚分流，月底結算時不需要員工自己回報。

沒有網路可以打卡嗎？

可以。ClocDot 是 PWA 應用，安裝到手機桌面後支援離線打卡，連線恢復時會自動同步打卡紀錄到雲端，不會漏記。

跨時區的遠端團隊怎麼處理打卡時間？

ClocDot 以員工所在時區記錄打卡時間，月底 CSV 匯出可選擇以公司總部時區或員工本地時區呈現，方便跨國團隊的薪資計算與工時管理。
