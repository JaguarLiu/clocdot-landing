What is GPS clock-in

什麼是 GPS 打卡？

GPS 打卡（也稱為定位打卡、位置打卡）是利用智慧型手機內建的 GPS 模組， 在員工按下打卡按鈕時自動擷取當下的經緯度座標，並比對是否在公司設定的範圍內， 進一步判斷該次打卡是「公司打卡」「外勤打卡」還是「遠端打卡」。

相較於傳統的卡鐘打卡需要員工本人到場，或Wi-Fi 打卡需要連到公司網路， GPS 打卡的優勢是：

  • 不需要實體設備 — 員工手機即打卡機，公司省下卡鐘採購維護成本。
  • 適用各種場景 — 辦公室、外勤拜訪、工地、客戶端、在家上班都能打。
  • 地點可追溯 — 每筆打卡都有座標可查，發生爭議時有客觀依據。
  • 支援多據點 — 一家公司有多個辦公地點時，系統自動分辨員工在哪一處。

Use cases

GPS 打卡的五種典型場景

🏢 多據點企業

總部 + 分公司 + 倉庫 + 門市。員工跨點支援時，系統自動分辨是哪個據點，月報自動分流統計。

🛵 業務 / 外勤拜訪

業務代表跑客戶、外勤工程師到場服務。GPS 自動記錄拜訪地點，主管不用追、業務不用寫日報。

🚧 工地 / 工程

建築、土木、機電工程的工地人員。每天工地不同，GPS 打卡可記錄當天到了哪個工地、待多久。

🏠 在家工作 (WFH)

員工在家或咖啡廳工作。GPS 自動標記為 Remote，與公司打卡分開統計，方便混合辦公管理。

🚚 物流 / 巡迴服務

物流司機、設備維修、清潔服務、長照訪視。員工一天跑多個地點，每一次打卡都記錄座標與時間，月底可拉出完整動線。

Privacy

員工的隱私怎麼保護？

很多員工聽到 GPS 打卡會擔心被全天追蹤。ClocDot 的設計原則是：

✓ 只記錄打卡當下的座標

員工按下「打卡」按鈕的那一秒擷取一次 GPS，不會在背景持續追蹤。員工沒打卡，系統就不知道他在哪。

FAQ

關於 GPS 打卡 的常見問題

GPS 打卡的準確度有多少？

GPS 打卡在戶外與一般市區的準確度約 10–20 公尺，足以分辨員工是否在公司範圍內。室內或高樓密集地帶因衛星訊號被遮擋，可能誤差到 30–50 公尺，系統會自動結合 Wi-Fi 與基地台訊號輔助定位。

員工會擔心隱私問題嗎？

ClocDot 只在員工主動按下打卡按鈕的當下記錄一次 GPS 座標，不會持續背景追蹤。員工不打卡時系統不會知道員工在哪裡。所有座標資料僅供主管查閱該次打卡是否在公司範圍內。

可以設定多個辦公室地點嗎？

可以。ClocDot 支援設定多個辦公地點（總部、分公司、客戶端、工地等），員工到任一地點打卡時，系統會自動分辨並標記是哪個地點。每個地點可設定不同的打卡半徑（預設 100 公尺）。

員工沒有開定位權限怎麼辦？

員工首次打卡時系統會請求 GPS 權限。若員工拒絕授權，系統會記錄該筆打卡為「無 GPS 資料」並通知員工開啟權限。
